Các thiết lập Options bên dưới chỉ dành cho trình duyệt Firefox42 (Updated 07/07/2021)

  • LẦN ĐẦU SỬ DỤNG: Mở Menu Options trên trình duyệt ➡ Chọn Advanced ➡ Certificates ➡ Chọn Select one automatically.

    Xong rồi các bạn Click hết vào các Link đính kèm/ Link Tool để Trình duyệt xác thực 1 lần, về sau bấm Open không báo nữa 👉Nhấn F5 tải lại trang là xong!

  • Lưu ý bấm nút Open báo lỗi "This Connection is Untrusted": Click vào link phía trên mở Tab mới > I Understand the risk > Add Exception > Confirm Security Exception.

  • Để trình duyệt chạy nhanh hơn: Mở Options trên trình duyệt ➡ Chọn Advanced ➡ Dòng Cache Web Content ➡ Click Clear Now (Click nhiều lần đến khi về 0 bytes nhé).

  • Lưu ý nếu mở trang báo lỗi "Server not found": Mở Menu Options trên trình duyệt ➡ Chọn Advanced ➡ Chọn Network ➡ Chọn Settings ➡ Chọn Use system proxy settings.

  • Nếu lỡ tay tắt đi Tab vừa đóng mà muốn mở lại: Nhấn tổ hợp phím "Ctrl + Shift + t" để mở lại (Rất cần thiết nếu lỡ tắt mất Tab có số đt KH vừa đọc).
